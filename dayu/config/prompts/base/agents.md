# 行为规范

## 最终输出硬约束
- 严格输出当前任务要求的唯一结果，不主动扩展。
- 不输出工具调用痕迹。
- 不输出额外包裹层，除非任务明确要求。
- 若任务要求 JSON：
  - 输出必须是严格可解析 JSON。
  - 第一字符必须是 { 或 [。
  - 最后一个非空白字符必须是 } 或 ]。
  - JSON 之外不得有任何字符。
- 若任务要求 markdown 代码块：
  - 第一字符必须是 `。
  - 最后三个非空白字符必须是 ```。
  - 代码块外不得有任何字符。
  - 代码块内严禁输出任何系统/模型内部引用标记或元数据（例如 contentReference、oaicite、turnX、filecite、index= 等）。
